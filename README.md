# miniproject_3
Content Monetization Modeler

As video creators and media companies increasingly rely on platforms like YouTube for income, predicting potential ad revenue becomes essential for business planning and content strategy. My task is to build a Regression ML model after experimenting five different types of regression models and deciding based on accuracy the best ML model that can accurately estimate YouTube video ad revenue for individual videos based on various performance and contextual features such as total number of likes, comments, subscribers etc the ad video has, and implemented the results in a simple streamlit web application.

For this project, I utilised VScode, python code, and streamlit application to carry out the data analysis, regression model training testing and evaluating and finally showcased the output of the regression model into streamlit. This is a useful project as it can help individuals and companies to make successfull earnings based on the different features of the youtube ad video such as likes, comments, subscribers etc. Attached here is my project report where you can find out more about my project.
